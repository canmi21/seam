# demo-client-vanilla

Vanilla JavaScript client demonstrating `@canmi/seam-client` for RPC calls and subscriptions.

## Structure

- `src/index.ts` — Client entry using `createClient()`
- `src/generated/` — Auto-generated typed client (output of `seam generate`)

## Development

- Generate client: `seam generate`
- Serve: open `index.html` or use any static server
