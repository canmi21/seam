# demo-server-bun

Standalone Bun server demonstrating `@canmi/seam-adapter-bun` with procedures, subscriptions, and pages.

## Structure

- `src/index.ts` — Server entry using `serveBun()`
- `src/procedures/` — Procedure definitions
- `src/subscriptions/` — Subscription definitions
- `src/pages/` — Page definitions with loaders

## Development

- Run: `bun run src/index.ts`
