# demo-client-react

React client demonstrating `@canmi/seam-react` with route definitions and server data hooks.

## Structure

- `src/main.tsx` — App entry point
- `src/routes.ts` — Client route configuration via `defineRoutes`
- `src/pages/` — Page components using `useSeamData` and `useSeamSubscription`

## Development

- Dev: `bun run dev`
