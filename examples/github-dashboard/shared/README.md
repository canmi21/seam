# @github-dashboard/shared

Shared React components and TypeScript types used by all [GitHub Dashboard](../README.md) frontends.

## Structure

- `components/` — Reusable UI components (`app-layout`, `profile-header`, `repo-card`, etc.)
- `types.ts` — Shared type definitions
- `styles/` — Shared CSS

## Notes

- No build step — consumed directly via workspace imports
